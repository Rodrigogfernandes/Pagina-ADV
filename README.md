## 📋 Sobre o Projeto

Site institucional moderno e responsivo desenvolvido para escritórios de advocacia, criado com HTML5, CSS3 e JavaScript puro (Vanilla JS). Esta página apresenta uma interface elegante e profissional, com design moderno que destaca os serviços jurídicos oferecidos e facilita o contato com clientes. O projeto foi desenvolvido com foco em performance, responsividade e experiência do usuário, utilizando apenas tecnologias nativas do navegador sem dependências de frameworks externos.

## ✨ Funcionalidades

- 🎨 **Design Moderno**: Interface elegante com gradientes e animações suaves
- 📱 **Totalmente Responsivo**: Adaptável a todos os dispositivos (desktop, tablet e mobile)
- ⚡ **Performance Otimizada**: Carregamento rápido e experiência fluida
- ♿ **Acessível**: Código semântico e suporte a leitores de tela
- 🔍 **SEO Friendly**: Estrutura otimizada para mecanismos de busca
- 📧 **Formulário de Contato**: Validação completa e integração com WhatsApp
- 🎭 **Animações Interativas**: Efeitos de scroll reveal e transições suaves
- 🎯 **Navegação Suave**: Scroll suave entre seções
- 📊 **Estatísticas Animadas**: Contadores dinâmicos de experiência e casos
- 🌟 **Seções Completas**: 
  - Hero com apresentação e call-to-action
  - Áreas de atuação (6 especialidades)
  - Sobre o advogado e credenciais
  - Processo de trabalho (4 etapas)

  - Formulário de contato

## 🎯 Áreas de Atuação

- **Direito Civil**: Contratos, responsabilidade civil, direito de família
- **Direito Empresarial**: Abertura de empresas, contratos comerciais, compliance e LGPD
- **Direito Trabalhista**: Ações trabalhistas, negociações coletivas
- **Direito Previdenciário**: Aposentadorias, auxílios e benefícios
- **Direito Criminal**: Defesa criminal, habeas corpus, queixa-crime
- **Consultoria Jurídica**: Análise de contratos, due diligence, assessoria contínua

## 🚀 Como Usar

### Pré-requisitos

Nenhum pré-requisito é necessário! O site é uma aplicação estática que pode ser aberta diretamente no navegador.

### Instalação

1. **Clone o repositório**
   ```bash
   https://github.com/Rodrigogfernandes/Pagina-ADV.git
   ```

2. **Navegue até o diretório**
   ```bash
   cd adv1
   ```

3. **Abra o arquivo `index.html` no navegador**
   - Opção 1: Clique duplo no arquivo `index.html`
   - Opção 2: Use um servidor local (recomendado):
     ```bash
     # Usando Python
     python -m http.server 8000
     
     # Usando Node.js (com http-server)
     npx http-server
     
     # Usando PHP
     php -S localhost:8000
     ```

4. **Acesse no navegador**
   ```
   http://localhost:8000
   ```

## 🎨 Personalização

### Cores
As cores principais podem ser alteradas no arquivo `assets/css/styles.css`. O site utiliza gradientes modernos:

```css
/* Exemplo de personalização de cores */
body {
    background: linear-gradient(135deg, #0f172a 0%, #1e293b 50%, #0f172a 100%);
}

.gradient-text {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### Conteúdo
- Edite o arquivo `index.html` para alterar textos, informações e links
- Substitua informações de contato (telefone, email, endereço)
- Atualize as áreas de atuação conforme necessário
- Personalize os depoimentos e estatísticas
- Modifique as credenciais e formação acadêmica

### WhatsApp
Para alterar o número do WhatsApp, edite o arquivo `index.html`:

```html
<!-- Botão flutuante do WhatsApp -->
<a href="https://wa.me/5583999999999?text=Olá,%20gostaria%20de%20agendar%20uma%20consulta">
```

E também no arquivo `assets/js/script.js` se houver integração adicional.

### Formulário de Contato
O formulário pode ser integrado com um backend ou serviço de email. Atualmente, a validação é feita no front-end. Para integração:

1. Configure um serviço de backend (PHP, Node.js, etc.)
2. Ou use serviços como Formspree, EmailJS, etc.
3. Atualize o JavaScript em `assets/js/script.js`
